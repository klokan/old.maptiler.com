Create own mobile app

A straight-forward way, how to create mobile apps usings maps rendered with MapTiler

Objective-C

For users of Objective-C for the development of a native iOS or Android application we have a ready-to-use free source code at:
https://github.com/klokantech/maptiler-ios-start

Follow these steps:

  1. Download the ZIP file
  2. Drop in MBTiles file rendered with MapTiler Pro
  3. Open in Xcode
  4. Compile

A distributable native app for iPhone/iPad is ready. Users can view and zoom the map and see their actual position.

JavaScript

For users of JavaScript for the development, then for native-like mobile apps is ideal jQuery Mobile:
http://jquery-ui-map.googlecode.com/svn/trunk/demos/jquery-google-maps-mobile.html
or Sencha touch:
http://dev.sencha.com/deploy/touch/examples/production/map/

The web app can be packed into a native mobile application for distribution via Apple AppStore / Google Play with http://phonegap.com/.
